◆ stateDef

enum stateDef

list of states for the main state machine

Enumerator
init 

Initialize all subsystems with status and error telemetry.

payloadMode 

Spacecraft normal operation (check TCs, send telemetry, compare sensor values).

processMsg 

Process the 5G packet(s) recieved (redirect to Intersat subsystem or to the ground station).

idleMode 

Activate the low power mode when no 5G packets have been recieved after a certain time.

processNav 

Process the navigation request to find the user location from the doppler shift algorithm

restart 

Free all subsystems variables, systemd script will restart the program when ending.

ending 

Stop the program.
