ADU * ADUSegments::unite ( )
inline

Join the pending ADU segments and produces a complete ADU. The complete ADU will be stored as a newly created instance of ADU. Deletion of the instance must be done in user application. The pending ADU segments (their instances) will be deleted inside this method.
