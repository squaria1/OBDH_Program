Detailed Description

A class that restores a complete ADU from ADU segments split into multiple CCSDS SpacePackets. This class was taken from HXI/SGD DataReceiver by Soki Sakurai and Hirokazu Odaka.

void ADUUnsegmenter::push ( CCSDSSpacePacket *  ccsdsSpacePacket)
throw (ADUUnsegmenterException
)
inline

Pushes a CCSDS SpacePacket to the unsegmenter. Pushed packet will be analyzed, and then connected to a pending packet list if it is a middle segment. If the segment is the last segment of a complete ADU, ADUUnsegmenter will join segments in the pending list, and pushes a unsegmented complete ADU to an internal buffer so that user application can retrieve the complete ADU via the popCompleteADU() method. Note: This method makes a copy of the input CCSDSSpacePacket instance, and pushes to a vector for unsegmentation. Deletion of the input CCSDSSpacePacket instance should be taken care outside this method.

Parameters
[in]ccsdsSpacePacketa CCSDS SpacePacket that contains an ADU segment
